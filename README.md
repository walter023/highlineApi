# Highline API

Description of the project

## How to install

Follow this steps:
- Clone repo
- Run `npm install`
- Run `npm start`

### `npm start`

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:300) to view it in the browser.
